# 推箱子游戏机 (Sokoban Arcade Machine)

基于 **Waffle Nano (Hi3861)** 主控的推箱子游戏机项目，使用 MicroPython 开发，通过摇杆控制角色推动箱子到目标点。

## 功能特性

- **5道递进难度关卡**：从1箱1目标入门到4箱对称布局综合考验
- **摇杆操控**：通过 JoystickShield 扩展板摇杆控制上下左右移动
- **目标点状态追踪**：PT（玩家在目标上）/BT（箱子在目标上）双状态标记，目标点永不丢失
- **实时步数显示**：屏幕顶部HUD显示当前关卡和步数，全部通关后统计总步数
- **关卡重置**：按 C 键一键重置当前关卡，无需重启
- **通关动画**：屏幕闪烁 + Clear! 提示，全部通关显示 You Win! 及总步数

## 硬件配置

| 组件 | 引脚 |
|------|------|
| ST7789 屏幕 (SPI0) | SCK=GP6, MOSI=GP8, DC=GP7, RESET=GP11 |
| 摇杆 X 轴 (ADC) | GP12 (ADC0) |
| 摇杆 Y 轴 (ADC) | GP13 (ADC1) |
| 按钮 A | GP14 |
| 按钮 B | GP0 |
| 按钮 C | GP1 |
| 按钮 D | GP2 |

## 运行方法

1. 将项目代码上传到 Waffle Nano 开发板
2. 确保已安装 `st7789` 驱动库
3. 将 `main.py` 作为主程序运行，或重命名为 `main.py` 自动启动

## 操作说明

| 操作 | 方式 |
|------|------|
| 移动角色 | 摇杆上下左右推动 |
| 重置关卡 | 按 C 按钮 |

## 元素编码

| 编码 | 含义 |
|------|------|
| E=0 | 空地 |
| W=1 | 墙壁 |
| B=2 | 箱子 |
| T=3 | 目标点 |
| P=4 | 玩家 |
| BT=5 | 箱子在目标上 |
| PT=6 | 玩家在目标上 |

## 开发环境

- **主控芯片**：Waffle Nano (Hi3861)
- **编程语言**：MicroPython
- **显示屏**：ST7789 (240x240)
- **输入设备**：JoystickShield 扩展板（摇杆 + 4按钮）
